Frequently Asked Questions

1) How can I change the font size of the messages?

There is a slide bar for changing the font size, using that slide bar you can easily change font size for your messages!

2) Can I have different color for each AI and User messages?

Sure! There are two different color palette for both User and AI messages. You can select different color for each of them.

3) Can I save a history for a specific date and send it to another person?

Sure! Using a calendar you can find the date that you want and by clicking on the fetch data button you can easily download your desired history and send it to anyone!
